  2. If your elements are still not
    exactly where you want them to be
    after adjusting the padding,
    margins and alignment, then you
    can try out the float property. The
    "float" property is one of the most
    powerfull tools you can master
    when learning CSS.

  1. Elements underneath a floating
    object will automatically wrap
    themselves around the content.
    For example, if you float an image,
    the text underneath will wrap
    around it so that none of it is
    actually obscured underneath the
    image.

  2. The float property liberates an
    element from its automatic
    position and lifts it up to "float" on
    top of other elements in the
    direction you specify. You can
    specify float either right, left or the
    default of none.
